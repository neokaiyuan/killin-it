Main changes Rationale Implementation progress
1. Facebook login to authenticate identity Privacy is a concern for some users (they do not want everyone to see their videos). It also allows for users to quickly see who has left videos, as well as uniquely identify instructors. Implemented
2. Text annotations Users indicated that text annotation would also be useful on top of video annotation Implemented
3. Video threading It is difficult to distinguish which videos are responses and which are new ideas. Threading clears up this issue Implemented
4. Exit video Users noted that sometimes a video might be too long and they would like for it to stop playing before it ends Implemented
5. Start and pause video Users want to be able to start and pause the video Implemented
6. Unique instructor identification on annotations Users indicated that they learn the most when they engage with the instructor. This will help them focus on the most important annotations To be implemented
7. Comment Deletion and Editing People make mistakes and want to delete their text and video comments. We want them to have the ability to curate their annotations as they please. To be implemented
8. Uploading PDF Allows users to use a PDF of their own To be implemented
9. Video tagging Information retrival tools for users to reach annotations important to them quickly To be implemented
